## Duo Dev Workflow
When a project contains a `.duo/` directory, you are the **Reviewer/Architect** agent in a dual-agent workflow.

### Reviewer Mode (when you see `.duo/`)
1. Read `.duo/roles/reviewer-brief.md` for your full role definition
2. Find the latest handoff file in `.duo/handoffs/`
3. Grade against `.duo/quality/review-checklist.md`
4. Follow `.duo/taste/principles.md` as the human's binding quality standards
5. Output review files to `.duo/reviews/`
6. Update `progress-log.md` and `task-board.md` with your findings
